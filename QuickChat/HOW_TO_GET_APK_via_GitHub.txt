============================================================
  HOW TO BUILD QuickChat APK using GitHub Actions
  (100% Free — No Android Studio needed)
============================================================


STEP 1 — Create a GitHub account (if you don't have one)
----------------------------------------------------------
  Go to: https://github.com
  Click "Sign up" → follow the steps


STEP 2 — Create a new repository
-----------------------------------
  1. Click the "+" icon (top right) → "New repository"
  2. Repository name: QuickChat
  3. Set to: Public  ← important, free Actions need this
  4. Click "Create repository"


STEP 3 — Upload the project files
------------------------------------
  On the new repo page, click "uploading an existing file"

  Drag and drop ALL files and folders from this QuickChat folder:
    ├── .github/
    ├── app/
    ├── gradle/
    ├── build.gradle
    ├── settings.gradle
    ├── gradle.properties
    ├── gradlew
    └── gradlew.bat

  ⚠️  IMPORTANT: Make sure .github/workflows/build.yml is uploaded
      (the .github folder may be hidden — show hidden files first)

  Scroll down → click "Commit changes"


STEP 4 — Watch it build automatically
----------------------------------------
  1. Go to your repo page on GitHub
  2. Click the "Actions" tab  ← at the top
  3. You will see "Build QuickChat APK" workflow running
  4. Click on it to watch the live build log
  5. Wait 3–5 minutes for it to complete
  6. When you see a green ✅ tick → build is done!


STEP 5 — Download your APK
----------------------------
  1. Click on the completed workflow run
  2. Scroll down to the "Artifacts" section at the bottom
  3. Click "QuickChat-APK" to download it
  4. Extract the zip → you get "QuickChat.apk"


STEP 6 — Install on your Android phone
-----------------------------------------
  1. Send QuickChat.apk to your phone
     (WhatsApp, Telegram, Google Drive, USB cable — any way)

  2. On your phone:
     Settings → Apps → Special App Access
     → Install unknown apps → allow your file manager

  3. Tap the APK file → Install → Open

  4. Grant all permissions when asked:
     ✅ Camera, Microphone, Storage, Location, Notifications


============================================================
  REBUILDING AFTER CHANGES
============================================================

Every time you push ANY change to GitHub, it automatically
rebuilds the APK. Just go to Actions tab and download again.

You can also trigger a manual build anytime:
  Actions → Build QuickChat APK → "Run workflow" button


============================================================
  TROUBLESHOOTING
============================================================

Build failed (red ✗)
  → Click the failed job → read the error log
  → Most common: Gradle version mismatch
  → Solution: change distributionUrl in gradle/wrapper/gradle-wrapper.properties

.github folder not uploading
  → On Windows: show hidden files in File Explorer
    View → Show → Hidden items → then drag .github folder

App says "Install blocked"
  → Settings → Security → Install unknown apps → Allow

App blank screen on phone
  → Check your internet connection
  → Visit https://sdck.pythonanywhere.com in phone browser first
